## Fair Use Policy

PokéAPI is free and open to use. However, we will ban IP addresses that abuse this privilege. This API is used primarily for educational purposes, and we want to keep it free and open.

## Rate Limiting

There is a rate limit of 100 requests per IP address per minute. If you exceed this limit, your IP address will be banned for one hour. Please be respectful of our resources and cache your responses.
